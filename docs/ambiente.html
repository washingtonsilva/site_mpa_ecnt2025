Ambiente Computacional

Instalação e Configuração dos Softwares

Visão Geral

Esta página contém instruções para instalação e configuração dos softwares necessários para acompanhar a disciplina Econometria Aplicada à Finanças.

Softwares Necessários

1. R (Linguagem de Programação)

O R é a principal ferramenta que utilizaremos para análises estatísticas e econométricas.

Instalação: - Acesse https://cran.r-project.org - Escolha o sistema operacional (Windows, macOS ou Linux) - Baixe e instale a versão mais recente (recomendado: R 4.4.0 ou superior)

2. RStudio Desktop

O RStudio é o ambiente de desenvolvimento integrado (IDE) recomendado para trabalhar com R.

Instalação: - Acesse https://posit.co/download/rstudio-desktop/ - Baixe a versão gratuita do RStudio Desktop - Instale após ter instalado o R

3. Quarto

O Quarto é o sistema de publicação científica que utilizaremos para criar relatórios reprodutíveis.

Instalação: - Acesse https://quarto.org/docs/get-started/ - Baixe e instale a versão adequada para seu sistema operacional - O Quarto já vem integrado nas versões mais recentes do RStudio

4. Git (Controle de Versão)

O Git será utilizado para controle de versão e colaboração nos projetos.

Instalação:

  • Windows: Baixe em https://git-scm.com/download/win
  • macOS: Instale via Homebrew (brew install git) ou baixe em https://git-scm.com/download/mac
  • Linux: Use o gerenciador de pacotes da distribuição (ex: sudo apt install git)

5. Conta no GitHub

Utilizaremos o GitHub para compartilhamento de código e entrega de projetos.

Criação da conta: - Acesse https://github.com - Crie uma conta gratuita - Configure seu perfil com nome e foto

Pacotes R Essenciais

Após instalar o R e RStudio, execute os comandos abaixo no console do R para instalar os pacotes principais:

# Pacotes para manipulação de dados
install.packages(c("tidyverse", "dplyr", "ggplot2"))

# Pacotes para dados financeiros
install.packages(c("tidyquant", "quantmod", "PerformanceAnalytics"))

# Pacotes para econometria
install.packages(c("broom", "car", "lmtest", "sandwich"))

# Pacotes para relatórios
install.packages(c("knitr", "rmarkdown", "DT"))

# Outros pacotes úteis
install.packages(c("here", "janitor", "skimr"))

Configuração Inicial

Configuração do Git

No terminal ou Git Bash, execute:

git config --global user.name "Seu Nome"
git config --global user.email "seu.email@exemplo.com"

Teste da Instalação

Para verificar se tudo está funcionando:

  1. Abra o RStudio
  2. Crie um novo arquivo Quarto (File → New File → Quarto Document)
  3. Teste o código R básico:
# Teste básico
library(tidyverse)
data(mtcars)
ggplot(mtcars, aes(x = mpg, y = hp)) + 
  geom_point() +
  labs(title = "Teste de Instalação")
  1. Renderize o documento clicando em “Render”

Suporte Técnico

Em caso de dificuldades na instalação:

  1. Consulte a documentação oficial dos softwares

  3. Entre em contato durante o horário de aula ou por e-mail
Dica Importante

Instale todos os softwares antes da primeira aula prática para aproveitar melhor o tempo em sala. Em caso de problemas, traga seu laptop para que possamos resolver juntos durante a aula.
