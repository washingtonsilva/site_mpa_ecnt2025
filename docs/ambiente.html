Ambiente Computacional

Instalação e Configuração dos Softwares

Visão Geral

Esta página contém instruções para instalação e configuração dos softwares necessários para acompanhar a disciplina Econometria Aplicada à Finanças.

Importante

  • Tente instalar e configurar todos os softwares antes da primeira aula para aproveitar melhor o tempo em sala.

  • Utilizaremos o RStudio como IDE para a linguagem R e o sistema Quarto devido à sua simplicidade, mas é possível utilizar outros IDEs como Visual Studio Code, Positron, entre outros, caso prefira.

Softwares Necessários

  • Linguagem R 4.5.1

  • IDE RStudio

  • Sistema Quarto

  • TinyTeX

  • Git

    • Git Bash como Terminal do RStudio

  • Conta no GitHub

Linguagem R

Instalando a Linguagem R 4.5.1

  1. Acesse https://cloud.r-project.org/

  2. Selecione o sistema operacional ou, se usar Ms Windows, clique em Download R for Windows por exemplo.

  3. Na página seguinte clique em base

  4. Clique em Download R-4.5.1 for Windows

  5. Feito o download basta ir clicando em próximo/next até a instalação ser concluída.

RStudio

Instalando o IDE RStudio

  1. Acesse https://posit.co/download/rstudio-desktop/

  2. Em geral, o site detecta seu sistema operacional e disponibiliza o instalador adequado logo abaixo de 2: Install RStudio.

  3. Clique em Download RStudio Desktop for Windows caso seu sistema operacional seja Windows.

  4. Finalizado o download, basta clicar no arquivo e, em seguida, clicar em em continuar/avançar/next até o início da instalação.

Sistema Quarto

Instalando o Sistema de Publicação Quarto

  1. Acesse https://quarto.org/docs/download/

  2. Clique em Download Quarto CLI e, finalizado o download, instale clicando no arquivo.

  3. Finalizado o download, basta clicar no arquivo e, em seguida, clicar em em continuar/avançar/next até o início da instalação.

TinyTeX

TinyTeX (necessária para gerar arquivos PDF)

Para renderizar arquivos PDF com Quarto/R é necessário instalar a distribuição TinyTeX (distribuição LaTeX mínima).

Após ter instalado a linguagem R e o RStudi, no Console R do RStudio, execute:

# instala o pacote tinytex pelo console do RStudio
# ou utilize o menu Packages → Install Packages
install.packages("tinytex")

# instala a distribuição LaTeX mínima
tinytex::install_tinytex()   

 # verifica se está instalado corretamente
tinytex::is_tinytex()

Manutenção/atualização

tinytex::tlmgr_update()      # atualiza pacotes LaTeX do TinyTeX

Teste rápido

Crie um arquivo hello.qmd e renderize em PDF:

---
title: "Teste de Renderização PDF"
format: pdf
---

Renderize o arquivo clicando no botão Render do RStudio ou utilize o terminal Git Bash (é necesário estar no diretório do arquivo):

quarto render hello.qmd

Git

Instalando o Git no Windows

  1. Acesse o site oficial do Git: git-scm.com

  2. Clique em Download for Windows, exibido em um terminal.

  3. Após o download, execute o instalador.

  4. Siga os passos do instalador, mantenha as configurações padrão sugeridas até o item 5.

  5. Há uma excecão, quando chegar na janela de título Adjusting the name of the initial branch, marque a opção Override the defaul e escreva main no campo apropriado.

  6. Clique em Next nas próximas janelas e finalize a instalação.

Configurando o Git

Configurar o Usuário

  1. Abra o Git Bash (encontrado no menu Iniciar após a instalação).

  2. Configure seu nome de usuário:

git config --global user.name "SeuNome"
Configurar o email
  1. Configure seu e-mail:
git config --global user.email "seuemail@exemplo.com"
Verificando as Configurações
  • Para verificar suas configurações, use:
git config --list

Git Bash como terminal no RStudio

RStudio: usando o Git Bash como terminal (Windows)
  • O Git Bash é instalado junto com o Git for Windows. Se não aparecer na lista, reinstale o Git mantendo a opção Git Bash.
  1. Abra RStudio → Tools → Global Options → Terminal.
  2. Em New terminals open with, selecione Git Bash.
  3. Confirme e reinicie o RStudio.

GitHub

Criando uma Conta no GitHub

  1. Acesse github.com

  2. Clique em Sign up para criar uma nova conta.

  3. Preencha os campos solicitados:

    • Username: Escolha um nome de usuário.
    • Email: Insira seu endereço de e-mail.
    • Password: Crie uma senha segura.

  4. Verifique seu e-mail e siga as instruções para confirmar sua conta.
